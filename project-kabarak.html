Kabarak University ODeL

Customised Moodle platform for enhanced online learning experience and faculty workflow.

E-Learning Platform8 Weeks

Project Overview

Kabarak University’s ODeL platform needed enhancement to better serve their growing online student population. The project focused on improving UX for both students and faculty while maintaining Moodle’s powerful functionality — involving custom development, interface redesign, and performance optimisation.

Moodle DevelopmentCustom DevelopmentUI/UX DesignPerformance

Quick Facts

  • Role: Moodle Developer & UX Designer
  • Timeline: 8 weeks
  • Tools: Moodle, PHP, JavaScript
  • Focus: User Experience & Performance

The Challenge

The existing ODeL platform had critical issues affecting both student engagement and faculty efficiency. It needed to scale for a growing user base while maintaining performance and accessibility standards.

User Experience

Complex navigation and inconsistent interface design confused both students and faculty.

Performance Issues

Slow load times and poor mobile responsiveness significantly limited accessibility.

Faculty Workflow

Inefficient course management tools were hindering teaching effectiveness.

Design Process

01

Requirements Gathering

Collaborated with faculty and students to identify pain points and gather requirements for an improved platform.

02

System Architecture

Designed improved information architecture and user flows for both student and faculty interfaces.

03

Custom Development

Built custom Moodle plugins and themes to address specific institutional needs and improve UX.

04

Testing & Deployment

Conducted user testing, performance optimisation, and gradual rollout with continuous feedback integration.

Key Features

Performance Optimisation

Improved load times and responsive design for seamless access across all devices.

Enhanced User Dashboard

Streamlined interface for both students and faculty with personalised experiences.

Custom Plugins

Specialised tools for assignment management, communication, and progress tracking.

Results & Impact

The enhanced ODeL platform significantly improved the online learning experience, resulting in increased student engagement and faculty satisfaction.
